⚙ Setup & Settings

Connect the app to your Google Sheet so all phones share the same live database.

📋 The app works fully offline — it reads from local cache when no internet is available. Changes made offline will sync when you reconnect.
1
Open Google Sheets and create a new spreadsheet named Freezer Database.
2
Open the Apps Script editor (Extensions → Apps Script) and paste in the script from google-apps-script.js included with this app. Click Deploy → New deployment → Web app. Set access to Anyone. Copy the URL.
3
Paste your Apps Script Web App URL below.


4
Share this index.html file with everyone in the household — they open it in their phone browser and tap Add to Home Screen. No app store needed.
